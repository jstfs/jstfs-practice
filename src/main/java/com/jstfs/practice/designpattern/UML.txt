
UML: Unified Modeling Language, 统一建模语言

实现接口:	空心三角 + 虚线
继承类:	空心三角 + 实线

以下4种关系都属于耦合关系:

	依赖关系:
		虚线 + 箭头
		弱耦合关系,可以认为对象B作为对象对象A的某个方法的入参,对象B的变化会影响到对象A的行为.
	关联关系:
		实线 + 箭头
		强耦合关系,可以认为对象B作为对象A的某个成员变量,或者通过对象A的某个方法可以获得对象B.	

	聚合关系:
		空心菱形 + 实线 + 箭头
		一种"弱拥有"关系,即A对象可以包含B对象,但B对象不是A对象的一部分.比如,雁群和某只大雁,这只大雁脱离雁群之后,雁群还是雁群.
	组合关系:
		实心菱形(数字) + 实线 + 箭头(数字)
		一种"强拥有"关系,即A对象和B对象是整体和部分的关系.整体和部分的生命周期是一致的.比如,大雁和它的两个翅膀.
